    User&Pass Auth¶

    Proxy list can be generated from the following two addresses

    Official website-Proxy setup-User:Pass

    Dashboard-Proxy setup-Username:Password

    Step 1:

    In the top row, select the corresponding plan type.

    # Tab Plan Type Corresponding port
    1 Unlimited Unlimited 12000
    2 Residential Pay/GB 10000
    3 Residential Pay/IP 20000
    4 Residential Wallet 50000
    5 Static Residential Static Residential 30000
    6 Datacenter Datacenter 35000

    Step 2:

    Select a proxy location (if not, global random by default)

    Step 3:

    Select session type:

    Rotating IP indicates that the IP address is changed on each request.

    Sticky IP, need to select session (8 random numbers) and sessionTime (maximum 180 minutes) Sticky IP allows you to keep your exit ip unchanged for a specified time frame. If you need to manually change the egress ip address, change the eight-digit session value.

    Step 4:

    Select username and password

    By default, the system generates a sub-account name to choose from, or you can create a new sub-account and password from the dashboard.

    Dashboard ＞ Sub-User Settings

    Step 5:

    Select a host. You are advised to select the host closest to your location.

    Hostname Region
    global.711proxy.com Asia
    as.711proxy.com Asia
    us.711proxy.com North America
    eu.711proxy.com Europe

    Complete the above steps

    A curl command will be generated that you can test. Open the cmd window, paste it in, and press Enter to test whether the proxy is available.

    Finally, choose the output format and generate quantity, and then click Generate to copy the created proxy list to the clipboard, you can apply this to any application or tool, and then you can use the proxy.
