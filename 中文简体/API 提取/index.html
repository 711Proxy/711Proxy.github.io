    API 提取¶

    可以通过以下两个地址生成提取链接

    官网 提取页面

    个人中心 提取页面

    第一步：

    最上面一排，选择对应套餐类型。

    1. 不限量：不限量提取。
    2. 动态住宅：按GB付费提取，按IP数量付费提取，使用钱包余额付费提取。
    3. 静态住宅：
    4. 独享数据中心：

    第二步：

    选择对应区域（也可以不选，默认全球混播）

    第三步：

    选择协议, http 或 socks5 (网页，建议使用 http，其他场景建议 socks5)

    第四步：

    根据需求选择api数据格式，（如无特殊需求，默认即可）

    第五步：

    点击生成API链接，此时可能会有弹窗，询问是否添加白名单，（如果是自己使用，需要将自己的ip添加到白名单才能使用，如果不是自己使用，需要将使用人的ip添加到白名单）

    第六步：

    复制页面上生成的链接。粘贴到应用的指定位置。（指纹浏览器，或 chrome 插件）
