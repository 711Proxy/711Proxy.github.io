    API 獲取¶

    可以通過以下兩個地址生成 API 鏈接

    官網-代理設置-API

    儀錶板-代理設置-API

    第一步:

    在最上面一行中，選擇相應的計劃類型。

    序號 Tab 頁 套餐類型 對應主機端口
    1 不限量 不限量提取 12000
    2 動態住宅 按 GB 付費 10000
    3 動態住宅 按 IP 付費 20000
    4 動態住宅 錢包餘額 50000
    5 靜態住宅 靜態住宅 30000
    6 獨享數據中心 獨享數據中心 35000

    第二步:

    選擇代理位置(如果不選，則默認爲全局隨機)

    第三步:

    選擇協議：http 或 socks5 (web 頁面建議使用 http；其他場景推薦使用 Socks5)

    第四步:

    根據需要選擇 API 數據格式（如無特殊要求，選擇默認格式即可）

    第五步:

    點擊生成 API 鏈接，此時可能會彈出窗口，詢問是否添加白名單（如果自己使用，需要將自己的 IP 添加到白名單中才能使用;如果不是自己使用，則需要將使用人的 IP 添加到白名單中）

    第六步:

    生成鏈接後，您可以選擇“複製鏈接”或“打開鏈接”，打開鏈接即可查看生成的代理列表，然後應用到反檢測瀏覽器或其他指定位置。
