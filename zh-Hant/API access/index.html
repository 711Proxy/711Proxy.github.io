API access¶

API links can be generated from the following two addresses

Official website-Proxy setup-API

Dashboard-Proxy setup-API

Step 1:

In the top row, select the corresponding plan type.

# Tab Plan Type Corresponding port
1 Unlimited Unlimited 12000
2 Residential Pay/GB 10000
3 Residential Pay/IP 20000
4 Residential Wallet 50000
5 Static Residential Static Residential 30000
6 Datacenter Datacenter 35000

Step 2:

Select a proxy location (if not, global random by default)

Step 3:

Select protocol: http or socks5 (http is recommended for web pages; socks5 is recommended for other scenarios)

Step 4:

Select API data format as required (default if no special requirements)

Step 5:

Click the generate API link, at this time there may be a pop-up window, asking whether to add the whitelist (if you use it yourself, you need to add your own ip to the whitelist to use it; if you do not use it yourself, you need to add the user's ip to the whitelist)

Step 6:

after generating the link, you can choose Copy link or Open link, open the link to see the list of generated proxies, then apply to the anti-detect browser or other specified location.
